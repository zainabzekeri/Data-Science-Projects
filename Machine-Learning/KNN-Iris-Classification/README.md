# KNN Iris Classification Project

This project demonstrates the implementation of the K-Nearest Neighbors (KNN) classification algorithm using Python and Scikit-learn.

## Objective

To classify iris flower species based on their features and understand how KNN works in machine learning.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

## Concepts Covered

* Loading datasets using `load_iris()`
* Exploring the Iris dataset
* Sepal Length vs Sepal Width (Setosa vs Versicolor)
* Petal Length vs Petal Width (Setosa vs Versicolor)
* Train-Test Split
* Building a K-Nearest Neighbors (KNN) Classifier
* Making predictions and evaluating the model

## Learning Outcome

This project helped me understand the basics of supervised machine learning, data preparation, classification algorithms, and model evaluation using Scikit-learn.
